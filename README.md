## 📖 Overview | نبذة عامة

**MKS Sci-Fi Portfolio** is a cutting-edge, high-performance web application designed to showcase the professional journey of **Eng. Mostafa Khalid Sallam**. It features a state-of-the-art interactive 3D particle background, holographic UI components, and an immersive user experience, all while remaining fully installable as a Premium PWA.

**موقع ملف أعمال مصطفى خالد (MKS Sci-Fi Portfolio)** هو تطبيق ويب متطور وعالي الأداء صُمم لعرض المسيرة المهنية **للمهندس مصطفى خالد سلام**. يتميز بخلفية ثلاثية الأبعاد تفاعلية، ومكونات واجهة هولوغرافية، وتجربة مستخدم غامرة (Sci-Fi)، مع إمكانية تثبيته كتطبيق ويب متقدم (PWA).

### ⚡ 3. PWA 2.0 & Smart Shortcuts | تطبيق ويب متقدم واختصارات ذكية

- *تحديث شامل لملف Manifest مع إضافة اختصارات سريعة للمشاريع والمقالات والتواصل مباشرة من أيقونة التطبيق.*

### 🛰️ 4. Cyber-Scan Header Animations | تحريك العناوين السيبراني

- *العربية والإنجليزية: تبديل فوري للغة مع دعم كامل للاتجاه من اليمين لليسار.*
